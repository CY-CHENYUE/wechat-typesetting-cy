# 参考案例集（不是模板库）

八种**已验证的气质组合**，每张卡记录的是手艺：某种质感是由什么配色、字号层级、形状语言、组件形态做出来的。

用途只有三个：设计没把握时找灵感、用户点名某种现成风格时对照、成功设计的沉淀归档。
**禁止把任何一张卡的组合整套照搬到新文章**——每篇文章的设计必须按 SKILL.md Step 2 从内容现场推导；案例回答"这种质感怎么做"，不回答"这篇文章该长什么样"。

## 常见对应速查

**这张表不是选择器，是校验器**：你为文章现场设计完之后，对照一眼，确认没有明显违背直觉（比如给悲伤的文章配了荧光橙）。用户要"现成风格、快点出"时，也可以按它直接对照案例卡。

| 内容信号 | 常见气质组合 | 也常见 |
|---|---|---|
| 深度观点、评论、随笔，论证为主 | 蓝调杂志 | 极简黑白 / 商务蓝灰 |
| 产品发布、科技报道、数据密集、有时间线 | 暗夜科技 | 商务蓝灰 / 活力橙黄（轻快向） |
| 情感、文学、演讲稿、严肃长文 | 极简黑白 | 蓝调杂志 / 暖纸阅读 |
| 生活方式、美食、旅行、手作、人物故事 | 暖纸阅读 | 清新治愈 / 极简黑白 |
| 健康、心理、育儿、轻知识科普 | 清新治愈 | 暖纸阅读 / 极简黑白 |
| 财经、行业分析、职场方法、报告解读 | 商务蓝灰 | 蓝调杂志 / 学术线框 |
| 盘点、清单体、活动宣传、年轻向话题 | 活力橙黄 | 清新治愈 / 暗夜科技（科技盘点） |
| 教程、方法论、技术科普、论文解读、含代码 | 学术线框 | 商务蓝灰 / 极简黑白 |

校验时的两条直觉：

- **设计拿不准就往"少"收**：减颜色、加留白、降组件密度——朴素永远不违和，花哨经常翻车
- 气质混杂的文章，按"读者读完想留下什么印象"定调，不按题材关键词硬匹配；同一公众号连续排版保持视觉一致，除非内容气质差异明显

**交付要求不变**：报告设计方案（气质/版面策略/配色及理由）+ 两个备选气质方向各一句话。用户说"换成 X"时内容结构不动，重做设计重排。

## 案例卡

### 1. 蓝调杂志 blue-magazine（原 blue-minimal）

**气质**：理性、克制、纸质杂志的编辑感
**适合**：深度观点、评论、随笔、行业观察；**避免**：促销、轻松搞笑内容

| Token | 值 |
|---|---|
| 页面底色 | `#fefefe`（白，section 流即可） |
| 正文 / 标题 | `#1A1A1A` / `#000000` |
| 主强调 | `#0044FF`（电光蓝） |
| 辅助灰 | `#888888` |
| 水印色 | `rgba(0, 68, 255, 0.1)`（仅用于文字） |
| 字体 | `'Times New Roman', serif` |
| 字号 | 大标题 32 / 章节 22 / 小标题 16 / 正文 15 / 说明 13 |
| 行距字距 | 正文 1.75 / 0.5px，两端对齐 |
| 形状语言 | 直角、4px 实色分割线、大留白 |
| 装饰 | 蓝底白字头部 banner、56px 水印章节编号、英文小标签（letter-spacing 2-3px） |

**组件偏好**：水印编号章节头、蓝底金句卡、三栏信息卡（table）、目录卡、蓝色下划线强调
**成品示例**：`assets/examples/blue-minimal.html`

### 3. 极简黑白 mono-minimal

**气质**：安静、文学、把舞台完全让给文字
**适合**：情感文、演讲/书信、严肃长文、不确定时的安全默认；几乎不踩雷

| Token | 值 |
|---|---|
| 页面底色 | `#fefefe` |
| 正文 / 标题 | `#262626` / `#111111` |
| 强调 | 不引入彩色：用字重 + 浅灰底纹 `#F5F5F5` |
| 辅助灰 / 细线 | `#999999` / `#E0E0E0` |
| 字体 | `-apple-system, 'PingFang SC', sans-serif`（文学向长文可换 serif） |
| 字号 | 大标题 26 / 章节 18 / 正文 15 / 说明 13 |
| 行距 | 正文 1.8-2.0，段距偏大，留白是主要设计语言 |
| 形状语言 | 1px 细线、无卡片或极少卡片、居中「· · ·」分隔 |
| 装饰 | 章节用细线+小序号（01），金句独立成段居中 |

**组件偏好**：细线章节头、居中金句（无底色，上下留白）、浅灰引用块
**注意**：这个方向的质量全靠节奏——段落短、留白足、强调极少

## 取色与纪律

设计配色（SKILL.md Step 2 第 4 问）时的手艺与底线：

1. **从文章自己的世界里取色**：问一遍"这篇文章的意象 / 行业 / 品牌是什么颜色的？"——咖啡=棕、茶=青绿、微信=绿、金融=藏蓝、火锅=红、深夜电台=暖黄。取到的色做主强调 + 浅底 + 装饰
2. **装饰细节呼应内容**：标签文案、序号样式、分隔元素从文章主题里找语言（美食文的分隔可以是 ❋，科技文是细线，书信体可以居中小点）
3. 文章没有明确意象、或该公众号要保持视觉一致 → 用克制的中性配色（深灰 + 一个低调强调色），并在交付时说明
4. 任何配色都守住：**三色纪律**（正文色 + 主强调 + 至多一个辅助色）、**对比度**（正文级文字必须够黑/够白，强调色只点缀可放宽）、`wechat-compat.md` 全部硬规则（背景实色、纯白写 #fefefe……）

检验标准：**把这套配色挪到另一篇主题不同的文章上，应该明显不对劲**——对劲说明设计没贴着内容走。

成功的设计建议沉淀回本文件加一张案例卡，供以后找灵感；案例只启发，不复用。
